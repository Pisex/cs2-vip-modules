# [VIP] Rainbow Model
My Discord server - https://discord.com/invite/g798xERK5Y

Adds the ability for VIP players to put a colored model

In **groups.ini** add:
```
"rainbow_model" "1/0"
```

In **vip.phrases.txt** add:
```
	"rainbow_model"
	{
		"en"	"Rainbow model"
		"ru"	"Разноцветная модель"
	}
```
