# [VIP] FOV
My Discord server - https://discord.com/invite/g798xERK5Y

Allows VIP players to change their viewing angle

In **groups.ini** add:
```
"FOV" "100,90,80,70,60"
```
Values separated by commas

In **vip.phrases.txt** add:
```
	"FOV"
	{
		"en"	"Viewing angle"
		"ru"	"Угол обзора"
	}
	"FOV_Title"
	{
		"en"	"Select FOV"
		"ru"	"Выберите FOV"
	}
```
