# [VIP] Healthshot
My Discord server - https://discord.com/invite/g798xERK5Y

Adds the ability for a VIP player to receive healthshots each round

In **groups.ini** add:
```
"healthshot" "healthshot value"
```

In **vip.phrases.txt** add:
```
	"healthshot"
	{
		"en"	"Healthshots"
		"ru"	"Шприцы"
	}
```
