# [VIP] KillScreen
My Discord server - https://discord.com/invite/g798xERK5Y

Adds the ability for VIP players to see a colored effect on kills

In **groups.ini** add:
```
"killscreen" "1/0"
```

In **vip.phrases.txt** add:
```
	"killscreen"
	{
		"en"	"Kill Screen"
		"ru"	"Цветной экран убийства"
	}
```
