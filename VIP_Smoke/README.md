# [VIP] Colored Smoke
My Discord server - https://discord.com/invite/g798xERK5Y

Adds the ability for VIP players to have multi-colored smoke

In **groups.ini** add:
```
"smoke_color" "random or rgb color"
```

In **vip.phrases.txt** add:
```
	"smoke_color"
	{
		"en"	"Colored smoke"
		"ru"	"Цветной смок"
	}
```
